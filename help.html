FREQUENTLY ASKED QUESTIONS

 

1. How to get a particular selected option from a Multiple Choice Question and fill up in a question textbox/option textbox?

2. How to get all selected options from a Multiple Choice Question and fill up in a question textbox/option textbox?

3. How to get a selected option from a single choice and fill up in a question textbox/option textbox?

4. How to get a particular unselected options from a Multiple Choice Question and fill up in a question textbox/option textbox?

5. How to get all unselected options from a Multiple Choice Question and fill up in a question textbox/option textbox?

6. How to get all unselected option from a single choice and fill up in a question textbox/option textbox?

7. How to get text written in a blank choice and fill up in a question textbox/option textbox?

8. How to get make a field for others/other in the options?















Answers

1. If Delta-12-dep is enabled for the particular option/s, the key #fromDep_(QNO)_(SELPREF) will give you the result. E.G. #fromDep_1_2 will fetch the option with the 2nd selection preference from question no 1. If you want to populate it into the options, use the key#fromDep_(QNO)_ALL, which will populate the options with their corresponding codes, and questions skipping and enabled depedencies as set. E.G. putting #fromDep_2_ALL only in the option text will fetch all the selected options and put it in the current question's options without putting any options codes. If Delta-12-unDep also is enabled for the particular option/s, the key #from_(QNO)_ALL will fetch all the options and populate it in the current question's options without putting any options codes. E.G. #from_5_ALL, will fetch from the 5th question and populate it in the current question's options.
